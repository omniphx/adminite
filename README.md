## Development

1. `yarn install`
2. Create a salesforce client Id and secret
3. Create/update an `.env` to include `ELECTRON_WEBPACK_APP_SALESFORCE_CLIENT_ID` and `ELECTRON_WEBPACK_APP_SALESFORCE_CLIENT_SECRET`
4. Run `yarn dev`
